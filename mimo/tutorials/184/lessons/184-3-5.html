When we have many if statements for one scenario, we can make the code more clear by using a switch statement.

int num = 123;
switch (num)
{
 case1:
  cout << "Number 1";
  break;
 case2:
  cout << "Number 2";
  break;
 case3:
  cout << "Number 3";
  break;
 default: 
  cout << "Invalid input";
  break;
}
Number 1Number 2Number 3

The main idea is that our switch statement evaluates an expression, in this case num and accordingly runs code under the appropriatecase statement.
