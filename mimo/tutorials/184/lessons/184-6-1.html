In the last lesson we saw that the &amp; when put before a variable name will return the memory address of the variable. This is the address-of operator.

We can do the opposite using the * operator before the memory location. This is the dereference operator and will return the value at a memory location.

int num = 250;
cout << &num << endl;
cout << *&num;
0x70f8b2271bcc
250

The memory location isn't always useful, but the value at the location let's us do a lot more!
