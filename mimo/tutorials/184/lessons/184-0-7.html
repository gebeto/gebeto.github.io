If you want to write programs and snippets along this micro-course, it's a great to get a development tool like IntelliSense or NetBeans.

Open your development tool and create a new C++ project. Once you've filled in some details, you'll have a ready-to-go main.cpp file with a main() function.

#include<iostream>

intmain()
{
 /* Dump code here! */
 return0;
}

Nice work! In your tool, you'll also find a Build button to build, or compile, your program and a Run button to execute it.
