The CHECK constraint reviews the value that is being added to the column, and checks whether it meets the condition.

Let's say songs can't be less than 3 minutes long.

ALTERTABLE playlist(
  ADD CONSTRAINT length
  CHECKCONFIRM (length >= 3);

See that? This constraint prevents us from adding any value below 3 to our length column.

Not quite, we want to check if the condition is met!
