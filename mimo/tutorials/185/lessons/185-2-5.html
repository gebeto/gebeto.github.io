With an ORDER BY clause, we can sort a result set by any column.

Let's insert some more records into the family table, select them and sort the whole thing!

INSERTINTO family (id, name, age)
VALUES (3, 'Robin', 12),
  (4, 'Win', 12);
SELECT *
FROM family
ORDER BY id DESCname ASCage;
 id | name  | age 
----+-------+-----
  4 | Win   |  12 
  3 | Robin |  12
  2 | Mom   |  48 
  1 | Dad   |  50  id | name  | age 
----+-------+-----
  1 | Dad   |  50 
  2 | Mom   |  48 
  3 | Robin |  12
  4 | Win   |  12  id | name  | age 
----+-------+-----
  3 | Robin |  12 
  4 | Win   |  12 
  2 | Mom   |  48 
  1 | Dad   |  50 

See that? We can use the ASC and DESC keywords to specify an order. If we don't, the result set is sorted in ascending order.
