Data types are like guidelines that we give the columns, or fields, to follow so that they store the right kind of values.

Let's see if we can spot a few data types in the friends table!

SELECT nameageheight
FROM friends; name  
-------
 Robin 
 age  
-----
  42 
 height 
--------
   1.72 

Ah, yes! name is a TEXT-type field for strings, age is an in INT-type field for whole numbers and height can take decimal numbers of type FLOAT.
