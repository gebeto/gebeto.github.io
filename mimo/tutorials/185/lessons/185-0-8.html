Queries are statements that display, or return, results based on the parameters that we provide. These results come in the form of temporary tables.

Which of these statements is a query, then?

SELECTname
FROM friends;CREATEDATABASE adventure;

That's it! This query returns a temporary table with the name column from the friends table.

Doesn't CREATEDATABASE just, well, create a database? A statement isn't a query if it doesn't return results.
