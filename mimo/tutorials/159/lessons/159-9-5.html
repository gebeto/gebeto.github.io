We can also pass an array to a method.

voidsayHi(String[]String people) {
 String output = "";
 for (String p : people) {
  output += "Hi " + p + "! ";
 }
 System.out.print(output);
}

String[] myFriends = { "Ben",
 "Luke" };
sayHi(myFriends);Hi Ben! Hi Luke!
