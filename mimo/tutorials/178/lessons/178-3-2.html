What if two scenarios just aren't enough? Well, we can use the elseif keyword to bring in another condition.

var temperature = 20;
if(temperature > 100) {
 console.log("Boy, it's hot!");
} elseifif(temperature < 32) {
 console.log("Jeez, it's cold!");
} else {
 console.log("Pretty normal.");
}
Jeez, it's cold!

Look at you go! An else if statement allows us to evaluate another condition if the condition after that follows the if keyword is false.

While we can use they if keyword to create another if statement, it doesn't make a lot of sense: its condition can only be met if the other condition isn't true.
