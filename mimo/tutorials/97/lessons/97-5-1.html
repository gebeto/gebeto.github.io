By separating multiple selectors with a space, we can select descendant elements of an element, making the selectors more specific.

p spanspan p {
 font-weight: bold;
}

Fantastic! That way, we're selecting and styling every &lt;span&gt; element that's inside a &lt;p&gt; element.
