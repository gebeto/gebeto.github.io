There are properties that allow us to control the height and width of block-level HTML elements.

.box {
 border-style: solid;
 width: 100%;
 height: 50px100%;
}
<!doctype html>
<html>
 <head>
  <linkrel="stylesheet"type="text/css"href="styles.css"/>
 </head>
 <body>
  <div class="box"></div>
 </body>
</html>
<html>
 <head>
  <style>.box {border-style: solid;width: 100%;height: 50px;}</style>
 </head>
 <body><div class="box"></div>
 </body>
</html>
]]>

Of course! We can set the dimensions of block-level HTML elements with the width and height properties.

Relative heights are tricky, as they set the height in percent of the HTML element's parent, which, in this case, is the body with a height of 0px.
