Throughout this micro-course, we'll work on a little project. By the end of it, we'll have a, well, somewhat stylish web page.

Create a web page with a heading and a paragraph and save it as index.html. Create an external style sheet, save it as styles.css and include it in the page.

<!doctype html>
<html>
 <head>
  <title>Henry's web page</title>
  <linkrel="stylesheet"type="text/css"href="styles.css">
 </head>
 <body>
  <h1>Henry's web page</h1>
  <p>This is the home page of what might become a web site some day.</p>
 </body>
</html>
