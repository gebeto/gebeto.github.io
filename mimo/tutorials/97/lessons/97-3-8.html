By setting the width and height of an element, we're actually setting the size of its content. The total dimensions include its padding, borders and margins.

p {
 background-color: #44b3c2;
 width: 150px;
 height: 100px;
 padding: 25px;
 border: 25pxsolid#f1a94e;
 marginmargin-top: 25px 0px;
}
<!doctype html>
<html>
 <head>
  <linkrel="stylesheet"type="text/css"href="styles.css">
 </head>
 <body>
  <p>I'm the actual content.</p>
 </body>
</html>
<html>
 <head>

 </head>
 <body><p>I'm the actual content.</p></body>
</html>
