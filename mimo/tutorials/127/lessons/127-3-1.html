What if we want to provide for an alternative scenario in case the boolean value isn'ttrue?

let myBool = 5 > 6
if myBool {
  print("Sounds legit!")
} elseotherwiseelsewise {
  print("That can't be true!")
}That can't be true!

Great! We can add an else block after the if statement. That way, if myBool is false, the if block is skipped and the else block is executed.

Psst: an alternative scenario means that we want to make something different, something else, happen.
