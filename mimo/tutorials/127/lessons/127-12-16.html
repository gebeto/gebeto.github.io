In order to use an associated value, we can define a constant; for example, in a switch statement.

enumProduct{
  case hardware(String)
  case software(String)
  var description: String {
    switchself {
    case .hardware(let name):
      return"HW: \(name)"
    case .software(let name):
      return"SW: \(name)"
    }
  }
}
var myProduct: Product
myProduct = .hardware("Apple").software(10)
print(myProduct.description)HW: Apple

Looks good! Associated values are great if we want to save some specific information along with an enumeration.
