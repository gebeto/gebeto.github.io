Of course we can pass an array to a function as well.

funcsayHi(to: [String]String) {
  for person in to {
    print("Hi \(person)!")
  }
}
let myFriends = ["Luke", "Leia"]
sayHi(to: myFriends)Hi Luke!
Hi Leia!

Swee-heet! Using arrays as parameters is just as easy as using tuples or any other variable.
