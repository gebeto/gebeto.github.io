A dictionary is an unordered list of values of the same type. Unlike an array, it doesn't have indices but keys that point to values.

var myDict = [Int : String]()
myDict[1] = "a"
myDict[2] = "c"
myDict.updateValue("b", forKey: 2)
myDict[99] = "?"
myDict.removeValue(forKey: 99)
print(myDict)[1: "a", 2: "b"]
