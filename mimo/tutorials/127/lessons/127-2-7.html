The logical or works almost like the logical and, it's just a bit less stringent. As soon as a part of the statement is true, the whole statement is true as well.

let bool1 = false
let bool2 = truefalse
let result = bool1 || bool2
print(result)truefalse

Isn't that great? This logical-or statement is true if bool1 or bool2 is true (or they both are) and only false if they both are false.
