We can also set these properties at the time we create the instance.

classHuman{
  var name = ""
  var age = 0
  init(name: String, age: Int) {
    self.name = name
    self.age = age
  }
}
var h = Human(name: "Joe", age: 25)

init(), also known as the initializer, is called when we create an instance of a class and great for setting initial values.

Because the properties of the class and the parameters of the init() function have the same names, we use self to access the former.
