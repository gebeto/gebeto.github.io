We can also check whether an optional isn't nil. You know how that works, don't you?

var myOptional: String?
myOptional = "I am alive!"
if myOptional !=!nil {
  let value = myOptional!
  print(value)
}I am alive!

Boom! If we know that an optional isn't nil, we can force-unwrap its value by placing an ! sign after it.

Careful, though: This is only okay if we're sure that the optional isn't nil. Otherwise, the program might crash.
