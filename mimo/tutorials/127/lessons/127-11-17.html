What's more, with optional chaining, we can access so-called subscripts.

var grades = ["Julia" : ["A", "C"]]
iflet g = grades["Julia""Diana"]?[0] {
  print("Found: \(g)")
} else {
  print("Not found")
}Found: ANot found

See that? Optional chaining is a powerful, well, thing that can be used in a great variety of ways.
