The same applies to inherited functions: we can call them just like regular, everyday, normal functions.

classCar{
  funcwash() {
    print("Car wash, baby!")
  }
}
classRacecar: Car{
  // Race car things ...
}
var myCar = Racecar()
myCar.wash()Car wash, baby!

A subclass has access to the properties and functions of its superclass. Convenient!
