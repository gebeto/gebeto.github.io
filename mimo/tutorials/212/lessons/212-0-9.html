If you're on macOS, it's a great idea to use the Terminal app every now and then in order to familiarize with it.

$ cd /Applications/Utilities
$ ls
...
Terminal.app
VoiceOver Utility.app
X11.app

The Finder can help you locate it in the Utilities folder in the Applications folder.

Psst: if you don't have Mac, you can also use an online environment like Cloud 9.
