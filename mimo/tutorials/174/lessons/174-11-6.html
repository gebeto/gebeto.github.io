Sometimes we have to access nested namespaces. The List class, for example, is defined in System.Collections.Generic.

using System.Collections.Generic;

namespaceApp
{
 classProgram
 {
  publicstaticvoidMain(string[] args)
  {
   var n = new System.Collections.Generic.ListList<int>();
   n.Add(8);
   n.Add(1);
   n.Add(5);
   string s = String.Join(", ", n);
   Console.Write(s);
  }
 }
}
8, 1, 58, 1, 5

See that? With the using statement in place, we can but don't have to use the List class's namespace hierarchy when we use it.
