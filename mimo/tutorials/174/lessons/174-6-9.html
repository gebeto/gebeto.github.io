Here's another method we'll recognize: it finds the index of a specific value in a list.

var speeds = new List<int>();
speeds.Add(37);
speeds.Add(29);
speeds.Add(6);
speeds.Add(29);
var i = speeds.IndexOf(37296);
Console.WriteLine(i);
012

See that? IndexOf() finds the first occurrence of the value and returns its index in the list. As always, these indices start at 0;
