The or operator checks if at least one of the bool-type values is true. If that's the case, the operator returns true.

bool b1 = 10 <> 11;
bool b2 = 23 > 9000;
bool result = b1 || b2;
Console.WriteLine(result);
truefalse

Boom! The || operator returns true as soon as one of the two bool-type values is true. Only if both of them are false does it return false.
