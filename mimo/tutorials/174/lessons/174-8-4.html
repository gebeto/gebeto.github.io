A foreach loop is probably the simplest of loops! We don't need to specify any index places. It loops through every element in a collection in order.

string[] pokemon = { "Bulbasaur", "Charmander", "Squirtle" };
foreach(intstring name in pokemon)
{
  Console.WriteLine(name);
};
