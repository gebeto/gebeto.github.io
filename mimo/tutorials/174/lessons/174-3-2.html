When the if statement returns false, we can also run a block called elseif, which will run when its condition is met.

int x = 1505010;
if (x > 100)
{
 Console.WriteLine("x > 100");
}
elseif (x > 30)
{
 Console.WriteLine("x > 30");
}
else 
{
 Console.WriteLine("x <= 30");
}
x > 100x > 30x <= 30

Sweet! We can have many elseif blocks but just a single else block.
