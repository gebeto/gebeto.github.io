So, int-type variables hold whole numbers. Sometimes we might want to use decimal numbers, though. For that we can use the double type.

int myInteger = 2;
double myDouble = 2.52; 
Console.WriteLine(myDouble);
2.5

Awesome! The myInteger variable only takes integer values such as 2. The myDouble variable will only take double values such as 2.0.
