"Agent Smith" is busy defeating the plans of loads of villains. In fact, he's defeating so many that it's hard to keep count.

Use type inference to create a plansDefeated variable. Start at 0, add 5, and another 1. Finally, display a summary in the console.

var plansDefeated = 0;
plansDefeated += 5;
plansDefeated++;
Console.WriteLine("Agent Smith defeated " + plansDefeated + " plans");
Agent Smith defeated 6 plans

Nice! We can save the count in an int-type variable and use the ++ code to increment the count.
