Double quotes, on the other hand, can handle amazing things like interpolation. We can make calculations or insert variables into strings. Wild!

shoes = "sandalsboots"
wear = "I wear #{1 + 1}#{shoes}!"
print wear
I wear 2 sandals!I wear 2 boots!

See that? The #{} expression, which we call interpolation, allows us to insert almost anything into strings. However, it only works with double quotes ("").
