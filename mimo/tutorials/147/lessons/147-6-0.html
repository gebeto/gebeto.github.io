In this chapter, we'll explore some of the secrets of making a web page look nice or, well, nicer.

As it turns out, every HTML element has an attribute that we can use to change its looks.

<!doctype html>
<html>
 <body stylehref="">
  I'm still in between tags.
 </body>
</html>
<html><body style="">I'm still in between tags.</body></html>
]]>

Nice! The style attribute allows us to specify an inline style for the HTML element.
