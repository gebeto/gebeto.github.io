For the sake of completeness, let's also talk about including an audio file in a web page.

<!doctype html>
<html>
 <body>
  <audio src="https://getmimo.com/r/audio.mp3index.html" controls></audio>
 </body>
</html>
<html>
<body><audio src="https://getmimo.com/r/audio.mp3" controls></audio></body>
</html>
]]>

Fantastic! Apart from width and height, the &lt;audio&gt; has almost the same attributes as the &lt;video&gt; tag.

Whoops! Did I forget to mention that we can only use .mp3, .ogg and .wav files?
