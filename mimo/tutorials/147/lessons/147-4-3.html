Ordered lists have an attribute that allows us to define the type of its marker.

<!doctype html>
<html>
 <body>
  <ol type="A"type>
   <li>Harry</li> 
   <li>Hermione</li>
   <li>Ron</li>
  </ol>
 </body>
</html>
<html>
<body><ol type="A"><li>Harry</li><li>Hermione</li><li>Ron</li></ol></body>
</html>
]]>

Nice! If we set the type attribute to "A", the list items are numbered with uppercase letters.

Other possible types are lowercase letters ("a") and roman numbers ("I" and "i").

As there are several possible types, the type attribute can't be a boolean attribute.
