A great way to include videos in a web page is to use YouTube and the easiest way to get there is to add an inline frame.

<!doctype html>
<html>
 <body>
  <iframe width="420" height="315" srchref="https://www.youtube.com/embed/Vhh_GeBPOhs">
  </iframe>
 </body>
</html>
<html>
<body><iframe src="https://www.youtube.com/embed/Vhh_GeBPOhs"></iframe></body></html>
]]>

Fantastic! The &lt;iframe&gt; tag has very similar attributes as the &lt;img&gt; tag, which makes it really easy to use.

With the exception of a closing tag, the &lt;iframe&gt; tag is very similar to the &lt;img&gt; tag.
