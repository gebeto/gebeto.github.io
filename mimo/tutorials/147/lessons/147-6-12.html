Where's the mistake in this snippet of HTML code?

<!doctype html>
<html>
 <body style="font-family:"times new roman";">
  <div>
   <h1>Welcome to my web page!</h1>
   <p>This web page is about my <spanstyle="color:green;">life</span>.</p>
  </div>
 </body>
</html>
The font-family attribute is written in double quotation marksThe CSS code in the &lt;span&gt; is wrongThe &lt;div&gt; doesn't have a style attribute

Of course! As the style attribute is enclosed in double quotation marks, the font-family value needs to be enclosed in single quotation marks.
