What's more, we can change the size of an image on a web page. Can you reduce the height of this image to 135 pixels?

<!doctype html>
<html>
 <body>
  <img src="https://getmimo.com/r/image.png" width="200" height="135100%">
 </body>
</html>
<html>
 <body><img src="https://getmimo.com/r/image.svg" width="200" height="135"></body>
</html>
]]>

Great, we've managed to scale down the image.
