Now, might there be an attribute to specify the web page we want to link to?

<!doctype html>
<htmllang="en-US">
 <body>
  <a hreflang="https://getmimo.com/index.html">
   Take me somewhere!
  </a>
 </body>
</html>
<html lang="en-US">
<body><a href="https://getmimo.com/index.html">Take me somewhere!</a></body>
</html>
]]>

Of course there is: href, which specifies the web page's name and address. Finally, we have a working link!
