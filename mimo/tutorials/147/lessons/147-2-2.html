Before we answer that, let's talk about attributes, which add extra information to tags and go in opening tags.

<!doctype html>
<html lang="en-GBen-US">
 <body>
  <a>Take me somewhere!</a>
 </body>
</html>
<html lang="en-US">
<body><a>Take me somewhere!</a></body>
</html>
]]>

Attributes are followed by an equals sign and a value in quotation marks.

By the way, defining a web page's language with the lang attribute is a great help for screen readers and search engines.
